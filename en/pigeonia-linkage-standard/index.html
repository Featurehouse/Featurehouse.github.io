Pigeonia Linkage Standard

Note: Other versions are for translation reference only, please refer to the Simplified Chinese version.

Pigeonia Linkakge Standard is a standard linkage of all Fabirc mods in Minecraft above 1.16+

It makes it easy for Fabric mods to link together. We strongly recommend every Fabric mods follow this linkage standard.

The newest version for the linkage is now v1.1.0。
