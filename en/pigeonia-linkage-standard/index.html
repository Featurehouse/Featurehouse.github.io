Pigeonia Linkage Standard

Every Linkage Standard is based on the Simplified Chinese version, other versions are only for translation reference only.

Pigeonia Linkakge Standard is a standard for making linkage in Minecraft Java Fabric 1.16+ mods.

It makes making linkage in Fabric mod more convenient. We suggest all Fabric mods follow this standard.

The latest version of Pigeonia Linkage Standard is v1.0.1.
